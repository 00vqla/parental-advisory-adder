## Requirements

- Python 3.6+
- Pillow library

## Installation

1. **Clone or download** this repository
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. **Run the application**:
   ```bash
   python3 adder.py
   ```
